OPERADORES EM JAVASCRIPT 

São sinais utilizados para manipular um valor ou para validar uma condição.
    
    TIPOS DE OPERADORES

        ATRIBUIÇÃO
             
             = : x = y, x é igual a y;
             += : x += y, x é igual a x + y;
             *= : x *= y, x é igual a x * y;
             /= : x /= y, x é igual a x / y;
             %= : x %= y, x é igual a x % y;

        ARITIMÉTICA

            + : soma;
            - : subtração;
            * : multiplicação;
            ** : exponencial;
            / : divisão;
            % : módulo;
            ++ : incremento;
            -- : decremento;
        
        COMPARAÇÃO

            == : igual a;
            === : mesmo valor e mesmo tipo;
            != : diferente;
            !== : valor e tipo diferentes;
            > : maior que;
            < : menor que;
            >= : maior ou igual;
            <= : menor ou igual;

        LÓGICA

            && : e
            || : ou
            ! : não

        CONDICIONAL

            If Ternário: (condição) ? expressão1 : expressão2
                ^ a primeira expressão é a instrução do if e a segunda expressão é a instrução do else.
